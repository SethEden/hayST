(inner) bootStrapBusinessRules() → {object}

Captures all of teh business rule functions string-to-function call map data in the rulesLibrary and migrates that data to the D-data structure. This is important now, because this data structure is how the plugin functionality will be imported and subsequently executed by the Haystacks platform.
