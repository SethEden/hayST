Module: loggers

Contains all of the functions necessary for logging to the console, and logging to the system-specified log file. Some of the logic here is duplicated from the Haystacks repo, ut kept super basic intentionally. Logging from the plugin will remain hard-coded, because the data isn't loaded, and by the time it is loaded its already useless to log. So uncomment the lines in the plugin code if you need to log from the plugin, and leave them commented out if you don't need to log. This is setup here so that the logs can e added to the application level log file, especially to include data, as the data can be too much and will easily overflow the console buffer.
